# React Real App - MERN

## Project ingredients:

<u>Client Side:</u>

- React
- Font Awesome Icons
- Bootstrap
- Axios
- Joi Validation
- JWT Decode
- React Toastify

<u>Server Side:</u>

- MongoDB
- Morgan Logger
- Lodash
- JWT
- Joi Validation
- Express - Web Framework
- CORS
- Config
- Bcrypt

---

### Instructions:

<u>Client Side:</u>

Installation: `npm i`
<br/>
Run application: `npm start`

<u>Server Side:</u>

Installation: `npm i`
<br/>
Run server development: `npm run dev`.
